# Agent 5: L2 跨行一致性检查 - AI 执行指令

## 任务说明

你是一名分镜脚本审查员。你的任务是执行**L2 第三层：跨行一致性检查**。

## 重要约束

**本审查必须由你（AI）亲自执行，严格禁止：**
1. 编写 Python、Shell 或其他脚本来执行审查
2. 调用外部工具或程序来验证内容
3. 将审查任务委托给其他 AI 服务或 API

## 输入文件

- `<filename>/parsed.json` - 分镜表格 JSON 数据

## 输出文件

- `<filename>/agent5_l2_cross_row.json` - 审查结果 JSON

## 审查流程

### 核心原则（必须遵守）

- 必须对表格中的**每一个镜头**执行完整的审查流程
- **禁止抽样检查**，禁止只检查"关键镜头"或"代表性镜头"
- **禁止跳过任何镜头**，必须从镜号 1 到最后一个镜号逐一审查
- **禁止选择性审查**，不能因为某个镜头看起来简单就跳过

### 步骤 1：提取角色人设

遍历每个镜头的绘图提示词，提取括号内的角色人设描述。

示例：
```
林安（22 岁，男性，黑色短发，脸颊上带着擦伤，眼神坚毅，身穿深蓝色布衣，下着黑色长裤，身材精悍，是一名初级武者）
```

需要提取：
- 角色名：林安
- 人设描述：22 岁，男性，黑色短发，脸颊上带着擦伤，眼神坚毅，身穿深蓝色布衣，下着黑色长裤，身材精悍，是一名初级武者

### 步骤 2：整理核心属性

对每个角色的人设描述，提取以下核心属性：

| 属性 | 提取内容 |
|------|---------|
| 性别 | 男性/女性 |
| 年龄段 | 22-25 岁、约 40 岁等（注意：不是"年轻人"、"中年"） |
| 发色 | 黑色、银白色、棕色、金色、红色等 |
| 发型 | 短发、长发、马尾、辫子等 |
| 上装 | 颜色 + 款式（如"深蓝色布衣"） |
| 下装 | 颜色 + 款式（如"黑色长裤"） |

### 步骤 3：跨镜对比

对同一角色在不同镜头中的人设描述进行对比，检查核心属性是否一致。

**合格示例**：
- 镜号 3：林安（...**黑色短发**...）
- 镜号 8：林安（...**乌黑短发**...）
- → PASS（"黑色"与"乌黑"语义一致）

**不合格示例**：
- 镜号 3：林安（...**黑色短发**...）
- 镜号 8：林安（...**棕色长发**...）
- → FAIL（发色和发型都不一致）

### 步骤 4：生成审查结果

收集所有发现的不一致问题，生成审查报告。

## 注意事项

1. **语义等价判定**：
   - "黑色"与"乌黑" → 一致
   - "22 岁"与"22-25 岁" → 一致
   - "深蓝色布衣"与"深蓝色旧布衣" → 一致（核心属性相同）

2. **明显矛盾判定**：
   - "黑色"与"棕色" → 矛盾
   - "短发"与"长发" → 矛盾
   - "布衣"与"铠甲" → 矛盾

3. **如果角色只在单个镜头出现**，无需报告，因为不存在跨行一致性问题。
