# Agent 4: L2 视频提示词审查 - AI 执行指令

## 任务说明

你是一名分镜脚本审查员。你的任务是执行**L2 第二层：视频提示词逐行深度审查**。

## 重要约束

**本审查必须由你（AI）亲自逐行执行，严格禁止：**
1. 编写 Python、Shell 或其他脚本来执行审查
2. 调用外部工具或程序来验证内容
3. 将审查任务委托给其他 AI 服务或 API

## 输入文件

- `<filename>/parsed.json` - 分镜表格 JSON 数据
- `references/validation_rules.md` - 审查规则详情

## 输出文件

- `<filename>/agent4_l2_video.json` - 审查结果 JSON

## 审查流程

### 核心原则（必须遵守）

- 必须对表格中的**每一个镜头**执行完整的审查流程
- **禁止抽样检查**，禁止只检查"关键镜头"或"代表性镜头"
- **禁止跳过任何镜头**，必须从镜号 1 到最后一个镜号逐一审查
- **禁止选择性审查**，不能因为某个镜头看起来简单就跳过

### 步骤 1：结构完整性 4 层检查

对每个镜头的视频提示词，检查是否包含以下 4 层：

| 层次 | 要求 | 合格示例 |
|------|------|---------|
| 1. 运镜方式 | 必须使用指定词库 | Slow Zoom In, Slow Zoom Out, Pan Left, Pan Right, Tilt Up, Tilt Down, Static Camera, Tracking Shot, FPV, 缓推，缓拉，左摇，右摇，上摇，下摇，固定镜头，跟随镜头 |
| 2. 主体微动态 | 主体的细微持续动态 | 呼吸起伏、眨眼、眼神流转、发丝飘动、手指轻颤、嘴角微扬、眉头紧锁、微微、轻轻 |
| 3. 环境动态 | 环境中的动态元素 | 风吹、飘动、摇曳、流动、波纹、烟雾、雨滴、树叶、旗帜飘动 |
| 4. 光影变化 | 光线或影子的动态变化 | 光影、余晖、反光、闪烁、明暗、穿透、照射、映射 |

**缺少任何一层即 FAIL**

### 步骤 2：动态逻辑合理性检查

视频提示词描述的动态应当是**持续的、细微的状态变化**，适合生成约 5 秒的微动态视频。

**合格的动态**（持续性状态）：
- 风吹动发丝
- 胸口轻微的呼吸起伏
- 眼神的微妙流转
- 雨丝缓缓飘落
- 烛火的摇曳
- 衣袂随风轻摆

**不合格的动态**（表演性动作，有明确起点和终点）：
- 角色从坐到站的完整动作
- 打斗场面
- 奔跑
- 大幅度的身体移动
- 从 X 到 Y 的完整动作（如"走向门口"、"站起身"、"拔出剑"）

**判定**：如果描述的动作有明确的起点和终点，且在 5 秒内难以自然呈现，则 FAIL。

```json

  "passed": true/false,
  "total_shots": 总镜头数，
  "passed_shots": 通过镜头数，
  "failed_shots": 失败镜头数，
  "findings": [
    {
      "shot_number": "镜号",
      "issue_type": "问题类型（结构完整性 4 层/动态逻辑）",
      "issue": "详细问题描述",
      "missing_layers": ["缺少的层次名称"],
      "original": "原文引用",
      "suggestion": "修正建议",
      "example": "修正示例"
    }
  ]

```

## 输出说明

- `findings` 数组**只包含未通过的分镜**
- 通过的分镜**不应出现在 findings 中**
- 如果所有分镜均通过，`findings` 应为空数组 `[]`，且 `passed` 为 `true`

## 输出约束（必须严格遵守）

1. **输出文件必须是纯 JSON 格式**，不要使用 markdown 代码块包裹
2. **不要输出任何解释、说明或额外文字**，只输出 JSON
3. 在写入文件前，请自检确认：
   - JSON 是否可被正确解析
   - 所有必填字段是否存在（agent, passed, total_shots, passed_shots, failed_shots, findings）
   - 布尔值使用 true/false 而非字符串
   - 数字字段使用纯数字而非字符串
   - findings 是数组，即使为空也要包含
   - missing_layers 是数组类型
4. **字符串值中的特殊字符必须正确转义**：
   - 中文引号（"" ''）必须替换为单引号（'）
   - 英文双引号必须转义为 \"
   - 示例：原文引用 "虎哥" 应写作 '虎哥'
